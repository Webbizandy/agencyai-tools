**Current Pattern**: "Tool Name - AgencyAI.tools"

**Examples**:
- Pipedrive: "Pipedrive Pricing 2025: How Much Does It Cost? ($14-$99/mo Breakdown)"
- ServiceTitan: "ServiceTitan Cost 2025: Real Pricing Per User ($200-$500/mo)"

### Meta Description Optimization
**Current Pattern**: Generic description

- "Pipedrive costs $14-$99/user/mo. Compare all 5 plans, see hidden fees, and learn what agencies actually pay. Updated Jan 2025 ‚≠ê 4.5/5 rating."
