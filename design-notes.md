# Design Inspiration from AI Tool Directories

## Futurepedia.io Key Design Elements:

### Color Scheme
- Bold blue gradient background (#4169E1 to deeper blue)
- Colored borders on category cards (green, pink, orange, red, blue)
- Each category has unique accent color
- Tool cards have subtle colored borders/badges

### Tool Cards
- Large logo/icon on left
- Tool name prominent
- Short description
- Colored tags/badges for categories
- Vote/like count with colored background
- Clean white cards with subtle shadows
- Hover effects

### Category Cards
- Colored left border (3-4px)
- Icon on left
- Category name
- Tool count badge in colored background
- Clean, organized grid layout

### Visual Elements
- Logo images for each tool
- Colored badges for pricing (Freemium, Paid, Free)
- Colored category tags
- Trust badges (Harvard, Nvidia, Meta logos)
- Gradient backgrounds
- Subtle shadows and depth

### Typography
- Large, bold headlines
- Clean sans-serif font
- Good spacing and hierarchy

## Design Improvements for AgencyAI.tools:

1. **Add colored borders to cards** - Each category gets unique color
2. **Tool card enhancements** - Add tool logos/icons, colored badges
3. **Category styling** - Colored left borders, better visual hierarchy
4. **Blog headers** - Add relevant images or gradient backgrounds
5. **Overall polish** - Better shadows, spacing, visual depth
6. **Logo fix** - Replace "App" with actual site name/logo
