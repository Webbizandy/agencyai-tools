## WRITING PROCESS PER TOOL

### Step 1: Research (10 mins)
- Check if Andy has used tool personally
- Look for real client results or case studies
- Verify current pricing (not estimates)
- Capture 2-3 screenshots from tool website

### Step 2: Write Andy's Take (20 mins)
Using this structure:

```
Opening Hook (2-3 sentences)
- Cut straight to opinion
- Lead with specific result or moment

Body (3-4 paragraphs)
- Real usage numbers and timeline
- What actually works in practice
- Client results with specifics
- White-label/agency angle if relevant
- Honest limitations (not just pros)

Quick Facts Box
- Bottom Line: Direct verdict
- My Setup: Exact configuration
- Client Results: Real numbers
- Worth it if: Specific criteria
- Skip if: Clear dealbreakers
```

### Step 3: Enhanced Content (15 mins)
- Replace sample FAQs with real customer questions
- Update pricing with actual tiers
- Add Andy's tips to feature descriptions
- Write agency-specific use cases

### Step 4: Screenshots (10 mins)
- Dashboard hero image
- 2-3 key feature screenshots
- Results/output examples if available

**Total per tool: ~55 minutes**
