## üêõ Troubleshooting

### Site Not Loading

1. Check Vercel **Deployments** tab - is the latest deployment successful?
2. Check **Domains** tab - is agencyai.tools showing "Valid Configuration"?
3. Try clearing your browser cache or using incognito mode

### Email Capture Not Working

1. Check Vercel **Environment Variables** - is `SYSTEME_API_KEY` set correctly?
2. Check Systeme.io - is your API key still valid?
3. Check browser console for errors (F12 ‚Üí Console tab)

### Changes Not Showing Up

1. Wait 2-3 minutes for Vercel to deploy
2. Hard refresh your browser (Ctrl+Shift+R or Cmd+Shift+R)
3. Check Vercel **Deployments** tab to confirm deployment succeeded
